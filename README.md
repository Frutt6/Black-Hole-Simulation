# Black Hole Simulation
2D (and maybe 3d in the future) simulation of how light bends around black holes.
Currently busy with adding more user input like changing the black hole's mass or letting them add light reflecting objects.
Long term goal is to use the GPU for the formulas that calculate the path of the rays.
